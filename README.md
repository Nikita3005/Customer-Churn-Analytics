# Customer-Churn-Analytics

Objective:
Analyze customer behavior to identify churn drivers and high-risk segments.

Tools:
Python, Pandas, Matplotlib, Seaborn, Power BI, DAX

Key Insights:

Higher churn observed among customers with short tenure

Month-to-month contracts show significantly higher churn rates

Long-term contracts correlate with improved retention

Output:
Interactive Power BI dashboard with KPIs and dynamic filters.
